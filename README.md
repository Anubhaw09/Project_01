<h3>Project Summary</h3>

This repository contains 34 self-contained Python programs implementing classic number-theory tasks, algorithmic building blocks, and mathematical utilities. The exercises range from beginner-friendly scripts (factorial, palindrome) to advanced algorithms (Miller–Rabin, Pollard’s Rho, Chinese Remainder Theorem, partition function). Each script is intended to show:

      Aim & brief description

      Methodology / algorithmic approach

      Example sample output (for quick verification)

      The project is designed for students learning algorithmic thinking and number theory implementation in Python.

<h3>Practicals (1–34) — Titles & Short Description</h3>

(These match the practical list from your assignment. Each script should contain a header with Aim, Methodology, and Example.)

1. factorial — compute n!

2. palindrome — check if integer reads same forwards/backwards

3. mean of digits — average of digits of a number

4. digital roots — repeated digit-sum to single digit

5. DIGITAL_ROOT(N) — (duplicate / same as 4)

6. Deficient Number — check if sum of proper divisors < n

7. Harshad Number — number divisible by sum of digits

8. automorphic number — square ends with the number itself

9. Pronic Number — product of two consecutive integers

10. List of Prime Factors — prime factorization

11. Count of distinct prime factors

12. Prime power — check if n = p^k for prime p

13. Mersenne prime — check 2^p - 1 primality for prime p

14. Twin primes — list twin prime pairs up to limit

15. Count divisors — number of positive divisors

16. Aliquot sum — sum of proper divisors

17. Are_Amicable — amicable pair check (e.g., 220 & 284)

18. Multiplicative_persistence — steps to single-digit by product

19. is_highly_composite — more divisors than any smaller number

20. mod_exp — modular exponentiation (fast)

21. Mod_inverse — modular multiplicative inverse (extended Euclid)

22. Chinese_Remainder — solve simultaneous congruences (CRT)

23. Quadratic_residue — Euler’s criterion for quadratic residues

24. order_mod — multiplicative order modulo n

25. fibonacci_prime — check Fibonacci number that’s prime

26. lucas_sequence — generate Lucas numbers

27. Perfect_power — check n = a^b for integers a>0, b>1

28. collatz_length — steps to reach 1 under Collatz map

29. polygonal_number — compute s-gonal numbers

30. is_carmichael — Carmichael number test

31. tesis_prime_miller_rabin — Miller–Rabin probabilistic primality test

32. pollard_rho — Pollard’s rho factorization algorithm

33. zeta_approx — approximate Riemann zeta function using series

34. partition_function — number of integer partitions of n
