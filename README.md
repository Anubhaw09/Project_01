Project Summary

This repository contains 34 self-contained Python programs implementing classic number-theory tasks, algorithmic building blocks, and mathematical utilities. The exercises range from beginner-friendly scripts (factorial, palindrome) to advanced algorithms (Miller–Rabin, Pollard’s Rho, Chinese Remainder Theorem, partition function). Each script is intended to show:

Aim & brief description

Methodology / algorithmic approach

Example sample output (for quick verification)

The project is designed for students learning algorithmic thinking and number theory implementation in Python.

How to Run

Clone the repository:

git clone https://github.com/Anubhaw09/Project_01.git
cd Project_01


Run a script (example):

python3 factorial.py


Each .py script is self-contained; run them individually. If your environment uses python instead of python3, replace accordingly.

Practicals (1–34) — Titles & Short Description

(These match the practical list from your assignment. Each script should contain a header with Aim, Methodology, and Example.)

factorial — compute n!

palindrome — check if integer reads same forwards/backwards

mean of digits — average of digits of a number

digital roots — repeated digit-sum to single digit

DIGITAL_ROOT(N) — (duplicate / same as 4)

Deficient Number — check if sum of proper divisors < n

Harshad Number — number divisible by sum of digits

automorphic number — square ends with the number itself

Pronic Number — product of two consecutive integers

List of Prime Factors — prime factorization

Count of distinct prime factors

Prime power — check if n = p^k for prime p

Mersenne prime — check 2^p - 1 primality for prime p

Twin primes — list twin prime pairs up to limit

Count divisors — number of positive divisors

Aliquot sum — sum of proper divisors

Are_Amicable — amicable pair check (e.g., 220 & 284)

Multiplicative_persistence — steps to single-digit by product

is_highly_composite — more divisors than any smaller number

mod_exp — modular exponentiation (fast)

Mod_inverse — modular multiplicative inverse (extended Euclid)

Chinese_Remainder — solve simultaneous congruences (CRT)

Quadratic_residue — Euler’s criterion for quadratic residues

order_mod — multiplicative order modulo n

fibonacci_prime — check Fibonacci number that’s prime

lucas_sequence — generate Lucas numbers

Perfect_power — check n = a^b for integers a>0, b>1

collatz_length — steps to reach 1 under Collatz map

polygonal_number — compute s-gonal numbers

is_carmichael — Carmichael number test

tesis_prime_miller_rabin — Miller–Rabin probabilistic primality test

pollard_rho — Pollard’s rho factorization algorithm
